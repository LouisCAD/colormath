CIE LCh(uv) (The cylindrical representation of CIE 1976 Luv*) color space.

l is a percentage, typically in the range [0, 100], but theoretically the maximum is unbounded. c is typically in the range [0, 230], but theoretically the maximum is unbounded. h is an angle in degrees, in the range [0, 360]
